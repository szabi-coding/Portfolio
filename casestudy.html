StartupCentral Redesign

Role UI Designer

Context Website Redesign

Period Early 2021

The Project

StartupCentral is a benefit club for entrepreneurs that focuses on client relationships and partner deals with other firms. The premise of the company is to create an online social platform where entrepreneurs can find help and information about starting and managing a business no matter whether they just want to start, or they have already started the process

Visit Website

An elegant design.

As the Front-end Developer, I was responsible of building the entire UI for the new website, redefining the User Experience and studying new interactions between the User and the Interface.

The process started by creating wireframes and UI components in Adobe Xd first with the home page, then the Blog and the SoMe page respectively.

For this project, a CSS Framework has been written in SASS for easier content creation.

On the previous website, the company received complaints how the site was not user friendly, furthermore people were unsubscribing from the service as they did not know, where they could potentially find relevant information or how they could register.

These issues have been adressed as now the userbase can find the register button easily thanks to the 'sizzling-red' colored register buttons, that gives a nice accent on the blue/white background colors.

The main 'space-cadet' blue button provides information through the site.

The Blog

StartupCentral is also a home for entrepreneurial news articles and blog posts. As part of the re-design the blog should contain the same colors so the userbase should be able to associate the different layout with the color scheme.

On the blog site a card layout welcomes the user to easily identify the different articles. The latest blog is bigger than the others and next to it a collection of the most popular and the staff favourites take place.

The topic of the blog is indicated by the 'sizzling-red' accent color and by clicking on it it takes the user to a collective page where only certain topics will show up.

The Blog article (picture on the right) features the blog post on the left side of the users' screen, and on the right shows the most popular and the staff favourites as well.

After the user reads the blog, 3 other blogs similar to the original are shown under the article to keep the user on the site. Under the three articles a category selector can be seen where each category is indicated with a different color.

Social Media Page

For the SoMe page a distinction was necessary as some of the users were complaining that they never knew whether they were logged in on the page or not. To make it clear for the user the logged in User Interface has been completely differentiated from the guest website.

Starting with the navigation bar, the color of the navbar has been changed to white and the initial text buttons have been changed to quick buttons with infographics on top.

This new navigation bar ensures that the content and help the user is looking for can be easily found and only the relevant buttons are shown for an easier access. If the user is lacking some information about a certain topic, the informative pages can still be found under the 'Menu' tab.

When the user is logged in the initial home page changes into an informative profile page. This page consists of 3 seperate columns. In the left row the user sees their name and a small welcome message, in the middle a newsfeed like column can be seen with concise information about what the user can and able to do by logging in, moreover the three latest blogs.
In the right column the user can get a quick glance of the various offers, the newest partners, and the newest coaches on the website.

Last but not least a user profile has been created (right image) as StartupCentral was planning on launching a Social Media platform for entrepreneurs.
On this page the user is able to cusomize their own profile by changing their profile picture, cover picture and add a descriptive information about themselves. Since the platform is made for entrepreneurs the profile page also lets the user to link their company with the user and write a descriptive information about the company itself, and also link their other SoMe profiles such as LinkedIn, Instagram etc.
